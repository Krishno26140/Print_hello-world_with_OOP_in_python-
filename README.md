# Print_hello-world_with_OOP_in_python-
This program demonstrates how Python uses classes, constructors, and controlled execution flow to run structured and scalable code
This program is not about printing "Hello World"

It teaches:

How objects are created
How constructors work
How programs are structured
How Python controls execution

Why is this important?

Because this structure is used in:

1. Real-world projects
2. APIs (Django/Flask)
3. Automation scripts
4. Large software systems
