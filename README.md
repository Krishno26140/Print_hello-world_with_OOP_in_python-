 Hello World using OOP in Python

A simple yet professionally structured Python program demonstrating the fundamentals of **Object-Oriented Programming (OOP)** along with clean execution practices.

---

 Project Overview

This project showcases how even a basic `"Hello, World"` program can be designed using **industry-standard coding structure**.
It focuses on writing clean, modular, and scalable Python code.

---

 Concepts Demonstrated

1. Class and Object Creation
2.  Constructor Method (`__init__`)
3.  Encapsulation (basic level)
4.  Function-based execution (`main()`)
5.  Controlled entry point (`if __name__ == "__main__"`)

---

 Code Structure

```python
class HelloWorld: 
    def __init__(self):
        print("Hello, World")

def main():
    obj = HelloWorld()   

if __name__ == "__main__":
    main()
```

---

 Why This Structure Is Used

 Object-Oriented Design

Using a class helps organize code into reusable and scalable components.
It reflects how real-world applications are built.

---

 Constructor Usage

The `__init__` method ensures that initialization logic runs automatically when an object is created.

---

 Clean Execution Flow

The `main()` function separates the program logic from execution, improving readability and maintainability.

---

 Safe Entry Point

The condition:

```python
if __name__ == "__main__":
```

ensures the script runs only when executed directly, preventing unintended execution when imported.
